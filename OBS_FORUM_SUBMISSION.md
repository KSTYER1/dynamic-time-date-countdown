## Overview

Dynamic Time/Date/Countdown is an unofficial third-party OBS Studio filter
bundle for text sources. It adds dynamic text filters for current time, current
date, and countdown displays.

## Features

- Dynamic time text filter.
- Dynamic date text filter.
- Dynamic countdown text filter.
- Focused text-source workflow.
- English and German UI text.

## Installation

Download the Windows x64 release archive and extract or copy its contents into
your OBS Studio installation directory.

The final layout should include:

```text
obs-plugins/64bit/dynamic-texts.dll
data/obs-plugins/dynamic-texts/locale/en-US.ini
data/obs-plugins/dynamic-texts/locale/de-DE.ini
```

Restart OBS after installation. The filters appear in the filter menu for text
sources.
